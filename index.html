Database Permission Denied

Your app is connected to Firebase, but it's blocking access. To fix this, update your Firestore Security Rules in the Firebase Console: 

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
